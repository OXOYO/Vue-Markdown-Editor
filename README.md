# vue-markdown-editor

> 基于 SimpleMDE && Vue2 封装的 Markdown 编辑器组件。

## 启动

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 资料
